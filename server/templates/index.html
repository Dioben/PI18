NEURAL NETWORK BEHAVIOR ANALYSIS TOOL DURING TRAINING

Easily monitor and experiment with neural networks

Features

A brief description of this project's features

Neural network data storage

The internal parameters of the neural networks run with our project are periodically stored in our database for future use and analysis.

Neural Network Analysis

The previously stored internal parameters can be accessed in an easily comprehensible format for ease of analysis.

Service Layer REST API

Our service layer is made with REST API for ease of use and integration with any other interfaces.

Web Application

We developed a web application in order to facilitate the use of our other features. It contains an easy way to deploy neural networks and a graph viewer to better analyse the progress of the deployed neural networks.
