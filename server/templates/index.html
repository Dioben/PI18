Features

A brief description of this project's features

Create a Simulation

Deploy a containerized Neural Network in the system using different formats including JSON and npz.

Neural Network Data Storage

The internal parameters of neural networks run with our project are periodically stored in our database for future use and analysis.

Simulation Result Analysis

The stored internal parameters can be accessed in an easily comprehensible format for ease of analysis including a variety of graphics.

Manage the Simulation Life-Cycle

Ability to pause or resume the Simulation process

Download Simulation Data

Download Simulation Data in csv format.

Web Interface

We provide a Web application in order to facilitate the use of the features.

Manage Users

A simple way for Administrators to manage users of the system
